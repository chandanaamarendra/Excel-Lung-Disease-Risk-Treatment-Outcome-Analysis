# 🫁 Lung Disease Risk & Treatment Outcome Analysis Dashboard (Excel)
## 📌 Project Overview

This project presents an interactive healthcare analytics dashboard built using Microsoft Excel to analyze respiratory disease trends, treatment patterns, and recovery outcomes.

The dashboard explores relationships between disease type, age group, smoking status, treatment methods, hospital visits, and geographic distribution across states.

## 🎯 Objectives

- Analyze respiratory disease prevalence

- Evaluate treatment distribution patterns

- Assess recovery rates across demographics

- Identify high-risk population segments

- Examine geographic variations in recovery outcomes

- Measure hospital utilization trends

## 🛠 Tools & Techniques Used

- Microsoft Excel

- Pivot Tables

- Pivot Charts

- Slicers

- KPI Cards

- Conditional Formatting

- Map Visualization

- Data Cleaning & Aggregation

## 📊 Key Metrics

- Total Patients: 11,085

- Total Hospital Visits: 85,621

- Average Age: 52.92 years

- Average Lung Capacity: 3.63

- Overall Recovery Rate: 51.32%

## 🔍 Key Insights

- COPD accounts for the highest disease share (46%).

- Individuals aged 50+ show significantly higher disease prevalence.

- Medication is the most commonly administered treatment across diseases.

- Average hospital visits per patient (~7.7) indicate chronic disease management burden.

- Recovery rates remain moderate (~51%), suggesting scope for improvement.

- Geographic variation in recovery indicates possible healthcare access disparities.

## 🚨 Potential Risk Indicators

- Higher disease concentration in older age groups (50+).

- COPD represents the dominant respiratory burden.

- High hospital utilization suggests chronic care dependency.

- State-wise variation highlights possible healthcare inequality.

- Moderate recovery rate indicates improvement opportunity.

## 📌 Dashboard Features

- Interactive slicers for dynamic filtering

- Smoking vs Recovery analysis

- Age-wise disease distribution

- Treatment utilization comparison

- State-wise recovery mapping

- Executive-level KPI summary
